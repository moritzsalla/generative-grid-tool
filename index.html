Custom Function

Type a custom function to calculate fill. Available variables:

  • x | x tile coordinate
  • y | y tile coordinate
  • c | c base color
  • noise() | simplex (perlin) noise
  • floor() | floor
  • ceil() | ceil
  • round() | round

Examples:
noise(x, y)
noise(x*c, sin(y))
noise(x+y, floor(sin(x+y)))
